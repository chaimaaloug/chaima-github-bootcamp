## Overview

Welcome to Chaima GitHub Bootcamp! This project is a presentation created with Spectacle to help you learn about Git and GitHub.

## Running the Presentation

1. **Install Dependencies:**
```bash
yarn install
```
or
```bash
npm install
```

2. **Start the Presentation:**
```bash
yarn start
```
or
```bash
npm start
```
